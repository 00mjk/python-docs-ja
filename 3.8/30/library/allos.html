汎用オペレーティングシステムサービス¶

本章に記述されたモジュールは、ファイルの取り扱いや時間計測のような (ほぼ)すべてのオペレーティングシステムで利用可能な機能にインタフェースを提供します。これらのインタフェースは、Unix もしくは C のインタフェースを基に作られますが、ほとんどの他のシステムで同様に利用可能です。概要を以下に記述します:

  • os --- 雑多なオペレーティングシステムインタフェース
    • ファイル名、コマンドライン引数、および環境変数
    • プロセスのパラメーター
    • ファイルオブジェクトの生成
    • ファイル記述子の操作
      • ターミナルのサイズの問い合わせ
      • ファイル記述子の継承
    • ファイルとディレクトリ
      • Linux 拡張属性
    • プロセス管理
    • スケジューラーへのインターフェイス
    • 雑多なシステム情報
    • 乱数
  • io --- ストリームを扱うコアツール
    • 概要
      • テキスト I/O
      • バイナリ I/O
      • Raw I/O
    • 高水準のモジュールインターフェイス
      • インメモリー ストリーム
    • クラス階層
      • I/O 基底クラス
      • 生ファイルI/O
      • バッファ付きストリーム
      • テキスト I/O
    • 性能
      • バイナリ I/O
      • テキスト I/O
      • マルチスレッディング

  • time --- 時刻データへのアクセスと変換

    • optparse からのアップグレード
